====== putty-plink ======
Created Wednesday 18 January 2017

To @connect (e.g. @push and @pull) to Github using an @rsa @key with @putty tooling from behind a @proxy:
1. Install //not// choosing **open ssh** but use the plink option instead.
2. Set Putty's **Default settings** to the proxy settings (as plink uses these).
3. Generate SSH key pair
4. Load private.ppk to Pageant.
5. Convert the public key to the **open ssh** format (using Putty's keygen).
6. Regsiter the result (public key in open ssh format) on github as an allowed key.
