# START/STOP FROM CLI

## start:
```
/usr/lib/vino/vino-server &
```

## stop:
```
user@system:~$ *pkill vino-server
```
   
## auto-start
Place in auto-start of DE:
```
/usr/lib/vino/vino-server
```


# AUTHENTICATION ISSUE: ""
```
gsettings set org.gnome.Vino require-encryption false
```

gnome-control-center
