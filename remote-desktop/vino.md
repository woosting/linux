# START/STOP FROM CLI

## start:
```
/usr/lib/vino/vino-server &
```

## stop:
```
user@system:~$ *pkill vino-server
```
   
## auto-start
```
/usr/lib/vino/vino-server
```
(in auto-start of DE)
